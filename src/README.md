OpsHub — Frontend-only enterprise demo

Run locally:

1. npm install
2. npm run dev

This project is deliberately frontend-only. All data is mocked and simulated in `src/mock/data.js`. No backend, DB, or real AI services are required. SMTP settings are stored locally and `Test Email` simulates sending.

Features:
 The app demonstrates invoice OCR simulation, reconciliation, document classification, approvals, AI Copilot, exception handling, RBAC, audit logs, notifications, and responsive UI.
- OCR simulation, Approvals, Exceptions, Admin settings
